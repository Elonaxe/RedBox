## 소개
RedBox는 샤오홍슈 크리에이터를 위한 로컬 AI 워크벤치입니다. 수집, 지식베이스, 원고 작성, 그룹 채팅, RedClaw 자동화를 통합합니다.

## 핵심 기능
- 내장 브라우저 + 원클릭 수집
- 로컬 지식베이스/검색
- 멀티 스페이스 분리
- Wander 아이디어 생성
- 원고 편집 + AI 보조
- 어드바이저 그룹 채팅
- RedClaw 단일 세션 자동화
- 이미지 생성/미디어 라이브러리 연동

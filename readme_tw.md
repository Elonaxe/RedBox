## 概覽
RedBox 是面向小紅書創作者的本地 AI 工作台，包含採集、知識庫、稿件、群聊與 RedClaw 自動化。

## 主要功能
- 內建小紅書瀏覽與一鍵採集
- 本地知識庫與檢索
- 多空間隔離
- 隨機漫步靈感生成
- 稿件編輯 + AI 輔助
- 智囊團群聊
- RedClaw 單會話自動化
- 生圖與媒體庫綁定

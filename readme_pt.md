## Visão geral
RedBox é uma bancada local de IA para criadores do Xiaohongshu. Integra captura, base de conhecimento, escrita, chat em grupo e automação com RedClaw.

## Recursos principais
- Navegador embutido e captura com 1 clique
- Base de conhecimento local e busca
- Isolamento por espaços
- Modo Wander para ideias
- Editor de manuscritos + IA
- Chat em grupo com advisors
- Automação de sessão única RedClaw
- Geração de imagens e biblioteca de mídia

## Resumen
RedBox es un banco de trabajo local de IA para creadores de Xiaohongshu. Integra captura, base de conocimiento, redacción, chat grupal y automatización con RedClaw.

## Funciones principales
- Navegador integrado y captura con un clic
- Base de conocimiento local y búsqueda
- Aislamiento por espacios
- Modo Wander para ideas
- Editor de manuscritos + IA
- Chat grupal con asesores
- Automatización de sesión única RedClaw
- Generación de imágenes y biblioteca de medios
